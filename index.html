📶 How QR Code Verification Works

When someone scans a QR code, they land on your site and the certificate is looked up from the built-in database in this HTML file. To make new certificates available to everyone:

Steps after adding/generating a certificate:
1 Go to Settings → Export Updated HTML
2 Download the index.html file
3 Upload it to GitHub / Vercel — replace the old file
4 Anyone scanning QR now sees the certificate ✓
