☁️ Cloud Database Setup

Certificates are stored in JSONBin.io — a free cloud database. Once connected, every certificate you create is instantly visible to anyone who scans a QR code, anywhere in the world. No re-deployment needed.

1
Go to jsonbin.io → Sign up free → Click "+ New Bin"
2
Paste this JSON as the bin content: {"certs":[]} → Click Create Bin
3
Copy the Bin ID from the URL (looks like: 6849abc123…)
4
Go to Account → API Keys → copy your X-Access-Key
5
Paste both below and click Connect
